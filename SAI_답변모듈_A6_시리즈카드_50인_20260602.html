A6 작가 시리즈 카드 · 50인 후보 데이터셋

인기 작가 50인의 공식 시리즈 보유 여부 + SAB 거래 데이터 텍스트 매칭 검증 결과. 작가 정보 답변(1번) 안에서 "이 작가의 시리즈" 목록을 미니 카드로 보여주기 위한 데이터셋.

← A6 답변모듈 목업으로 돌아가기

운용 방식

작가 답변 화면(A1·A2 모듈 하단)에서 "이 작가의 시리즈" 영역을 가로 캐러셀로 노출. 카드 1개 = 시리즈 1개. 카드 클릭 → A6 단독 시리즈 카드 답변으로 전환(시기 구분 + 시기별 미리보기 + 시리즈 전체 작품 M2 호출).

예시 답변 흐름: "이우환에 대해 알려줘" → 작가 답변(A1/A2) + 하단 "이우환의 시리즈" 캐러셀에 8개 미니 카드 (From Point · From Line · From Winds · Correspondence · Dialogue · Relatum · From Brush · A Cut Shot) → 사용자가 "From Point" 카드 탭 → A6 시리즈 답변 화면

A 등급 (즉시 적용)
18명

13명

총합
50명
  • A 등급 — 작가 공식 / 미술관·재단 / 메이저 경매사 lot description으로 시리즈명 학술 정착 + SAB DB title 텍스트 매칭으로 작품 자동 분류 가능. 별도 series 테이블 없이도 1차 적용 가능.
  • B 등급 — 작가 공식 시리즈는 있으나 SAB 거래 표본 매칭이 약하거나 (모두 Untitled 등), 큐레이터 작품-시리즈 매핑 사전이 별도로 필요한 작가.
  • C 등급 — 작가가 명명한 시리즈가 없거나(모티프·주제는 일관되나 시리즈명 부재), SAB 거래 표본이 너무 적은 작가. A6 미사용·M2(전체 작품) 폴백.
  • 표기 룰 정합 — 카탈로그 §AA-1148 영문 원제 표기 기본. 한글 표기는 작가 공식·미술관·정착 학술 표기 확인된 경우에만 병기. 임의 의역 금지.
  • 시기 구분(period_phases)은 큐레이터 정의가 1순위. 본 데이터셋은 학술/미술관 도록 정착된 시기만 표기. 미정의 시리즈는 단일 기간만 표기.

작가 공식 시리즈 정착 + SAB title 텍스트 매칭으로 자동 분류 가능. 별도 큐레이터 사전 없이 1차 적용 가능 (정확도 점진 보정).
