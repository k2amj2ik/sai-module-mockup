1. 기본 컨셉 — 이용 흐름 변경

답변 표시 방식을 "매칭 템플릿 + 재질문 버튼" 흐름으로 바꾼다.

  • 답변을 표시할 때 매칭되는 템플릿 형태로 표시한다.
  • AI 답변 하단의 재질문 버튼을 터치하면, 해당 재질문 유형에 맞는 템플릿 화면으로 이동한다.
  • 재질문 답변 템플릿 유형 (8종): 시세작품선택 · 시세결과 · 경매기록 · 경매순위 · 대표작 · 시리즈 · 화풍 · 1:1컨설팅

작동 방식 (2026-06-18 결정)

  • 작동형 시안: 재질문 버튼·작품을 실제로 누르면 폰 화면 안에서 해당 템플릿 화면으로 전환되고, 상단 ← 뒤로로 돌아온다.
  • 좌측 목록 구성: 진입 질문 8종(§6 질문별 기대 결과) + 템플릿 8종 단독 보기를 모두 노출한다.

2. 재질문 버튼 표시 조건

  • AI 답변 안에 등장한 작가의 작품에 대해 시세(혹은 시세작품선택) → 경매기록 → 경매순위 → 대표작 → 시리즈 → 화풍 순서로 관련 질문이 1개씩 표시된다. (버튼 6종, 순서 2026-06-18 확정)
  • 표시되는 질문은 최대 6개이다.
  • 작품 다수 시 시세·경매기록 버튼은 맥락별 분리 (2026-06-18 확정): 같은 작가의 작품이 여러 개면 대표 1작품에 대해서만 / 서로 다른 작품(모티프 검색 등)이 여러 개면 작품별 각각 (최대 6개 한도 내).
  • 작가는 특정되었으나 작품이 특정되지 않은 경우, 시세·경매기록 질문은 표시하지 않는다.

2-1. 시세 버튼

  • 표시 조건: 해당 작품에 경매 기록이 존재하여 시세 도출이 가능한 경우 표시. 터치 시 해당 작품의 시세 상세 화면으로 이동.
  • 단, 동일 작가/작품명 작품이 여러 개인 경우 작품 선택 화면이 먼저 표시된다.
  • 작품 선택 화면: 작품 썸네일 · 작가명 · 작품명 · 작품설명(제작년도, 카테고리, 사이즈, 재질)로 구성된 작품 정보가 목록 형태로 제공되어 터치 유도. 터치 시 해당 작품의 시세 상세로 이동. (작가·작품명이 같아도 카테고리·사이즈 등에 따라 시세가 달라지므로, 정확한 시세로 이동하기 위해 필요한 단계)
  • 질문 예시: [시세] '19-VI-71 #206' 작품 시세

2-2. 경매기록 버튼

  • 표시 조건: 해당 작품에 경매 기록이 존재하면 표시. 터치 시 해당 작품의 경매기록 템플릿으로 이동.
  • 질문 예시: [기록] '19-VI-71 #206' 최근 경매 기록

2-3. 경매순위 버튼

  • 표시 조건: 해당 작가에 경매 기록이 2개 이상 존재하면 표시. 터치 시 해당 작가의 경매순위 템플릿으로 이동.
  • 질문 예시: [기록] 김환기 경매 낙찰가 순위

2-4. 대표작 버튼

  • 표시 조건: 해당 작가에 연결된 대표작 데이터가 존재하면 표시. 터치 시 해당 작가의 작품 템플릿(대표작)으로 이동.
  • 질문 예시: [작품] 김환기 대표 작품

2-5. 시리즈 버튼

  • 표시 조건: 해당 작가에 연결된 시리즈 데이터가 존재하면 표시. 답변에서 구체적 작품이 언급된 경우 해당 작품에 가까운 시리즈를 표시. 터치 시 해당 작가의 시리즈 템플릿으로 이동.
  • 질문 예시: [작품] 김환기 달항아리 시리즈

2-6. 화풍 버튼

  • 표시 조건: 해당 작가에 연결된 화풍 데이터가 존재하면 표시. 터치 시 해당 작가의 화풍 템플릿으로 이동.
  • 질문 예시: [화풍] 단색화에 대해 알아보기

# 질문 기대 결과
4-1 김환기에 대해 알려줘 AI 답변 표시 + 하단에 시세 / 경매기록 / 경매순위 / 대표작 / 시리즈 / 화풍 순 추가 질문 버튼

4-4 파란색 장미가 나오는 작품 알려줘 AI 답변 표시 + 특정 작가·작품 여러 개 가능. 서로 다른 작품이 3개면 §2 맥락별 분리에 따라 3개 작품 각각에 대한 시세/경매기록 버튼 (최대 6개 한도)
4-5 단색화 작가는 누구야? AI 답변 표시 + 언급 작가 각각에 대해 시세/경매기록/경매순위/대표작/시리즈/화풍 순 버튼. 작가 우선 적용, 표시항목 우선순위 후순위 (작가 2명이면 A시세 → B시세 → A경매기록 … 라운드로빈)
4-6 작년에 제일 비싸게 팔린 작품 뭐야? AI 답변 표시 + 언급된 작가의 작품 각각에 대해 시세/경매기록/경매순위/대표작/시리즈/화풍 순 버튼
4-7 한국 탑 12 고가 거래 작품 (순위 의도 파악) → 경매순위 템플릿으로 바로 이동
4-8 작품을 팔고 싶어요 (판매 의도 파악) → 1:1컨설팅 템플릿으로 바로 이동
