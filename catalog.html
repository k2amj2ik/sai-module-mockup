SAI 답변 모듈 기획서

2026-06-17 기준 · Seoul Auction Intelligence (미술 자산관리 AI) · 답변을 재사용 모듈 조합으로 설계

1. 개요

SAI 답변은 AI 답변(자연어)을 기본 틀로 하고, 답변 단락에서 추출한 키워드와 매칭되는 데이터 모듈을 해당 단락 하단에 붙여 구성한다. 데이터 모듈은 서울옥션블루(SAB) 자체 경매 데이터를 근거로 하며 Gemini·ChatGPT 대비 핵심 차별화 영역이다. 9월 오픈 범위는 작품·작가·시리즈·경매까지이며, 시장통계·전시·뉴스 등은 이후 별도 산정한다.

2. 운영 모듈 (8종)

모듈역할
AI 답변모든 답변의 기본 틀(모듈로 보지 않음). 단락 생성 + 단락별 데이터 모듈 호출
작가언급된 작가 카드(대표작 썸네일 + 한줄설명). 1명만 있어도 100% 폭. 동명이인 있으면 함께 표시
작품언급된 작품 카드(작가명·작품명·캡션)
시리즈언급된 시리즈 카드(작가·시리즈명·시기·설명 + 썸네일)
화풍언급된 화풍 카드(시기·지역·대표 작가)
경매 기록단일·유사·최근 통합. 가로 스와이프 카드(최근 거래순, 최대 10건)
경매 순위순위·최고가 의도. 1개씩 큰 카드, 낙찰가 높은순, 1위 100% 바게이지
컨설팅 안내흰 박스 + "1:1 미술 컨설팅" 헤더 + 의도별 CTA. 배너 바로 위에 액션 버튼(복사·좋아요·아쉬워요·새로고침)

3. 단락별 키워드 → 모듈 매칭

AI 답변 단락에 실제로 언급된 대상만 모듈로 표시한다(기간·연관도로 임의 추정하지 않음).

단락 키워드매칭 모듈표시 규칙
작가명작가단락에 등장한 작가만. 동명이인 함께 표시
작품명작품단락에 등장한 작품만
시리즈명시리즈언급된 시리즈만 (시기로 추정 안 함)
화풍명화풍언급된 화풍만
경매·낙찰·기록경매 기록가로 스와이프 카드, 최근 거래순
제일·최고·탑N·순위경매 순위1개씩 큰 카드 + 낙찰가 높은순 + 1위 100% 바게이지

4. 질의 의도별 화면 구성

의도 (예시 질의)답변 구성
작가 정보
"김환기에 대해 알려줘"		AI 답변(소개) + 작가(언급 작가) + AI 답변(작업·화풍) + 화풍 + AI 답변(대표작) + 시리즈 + 작품 + AI 답변(발자취·경매) + 경매 기록 + 컨설팅 안내
작품 시세
"김환기 'Universe' 작품 시세"		모듈 미적용 — 기존 시세 화면: 시세 정보(AI 정확도) + 가격 추이 그래프 + 유사 작품 경매 기록(대표 1건 + 더보기) + 컨설팅 배너
작가의 작품들
"김환기 작가의 작품들"		AI 답변(시기별) + 작품(시기별) + 시리즈 + 컨설팅 안내
작품목록
"파란색 장미가 나오는 작품 알려줘"		AI 답변(작가·작품명 언급) + 작품(언급 작품) + 컨설팅 안내
작가목록
"단색화 작가는 누구야?"		AI 답변 + 화풍(언급 화풍) + 작가 + 컨설팅 안내
순위·최고가
"작년에 제일 비싸게 팔린 작품"·"한국 탑 12 고가 거래"		AI 답변 + 경매 순위 + 컨설팅 안내
컨설팅 유도
판매·보관·이동/설치·구매·기타		AI 답변(일반 안내) + AI 답변(서울옥션블루 서비스 안내, "작품 X 가이드"는 소제목) + 컨설팅 안내(의도별 CTA)

5. 공통 표시 정책

  • 단락 언급 기준: 모듈은 단락에 실제 언급된 대상만 표시(기간·연관도 추정 안 함)
  • 모듈 제목 "연관" 미사용: 작가 / 작품 / 시리즈 / 화풍 / 경매 기록 / 경매 순위 — 영역 인식용 작은 회색 볼드
  • 라운드 박스 제거: 답변 화면 모듈은 테두리 없이 하나의 답변으로 연속 표시
  • 1개 데이터 100% 폭 / 가로 스와이프 숫자 페이징("현재 / 총")
  • 컨설팅 배너 통일: 흰 박스 + "1:1 미술 컨설팅" 헤더 + 의도별 버튼, 배너 바로 위 액션 버튼
  • 출처·신뢰도 라벨: 데이터 모듈 "SAB 자체 경매 데이터 N건 기반" / AI 답변 "AI 일반 지식 기반"
  • SAB DB 미보유 작가: AI 답변 + 컨설팅 안내 강조 + "SAI 데이터가 부족합니다" 명시
