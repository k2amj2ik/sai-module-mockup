S-1 화풍 데이터셋 · 서양·한국·동양 근현대 30종

화풍(Style) 모듈 운용을 위한 초기 데이터셋. 서양 근현대 15종 + 한국 근현대 10종 + 일본·중국 근현대 5종 = 총 30종. 각 화풍별 시기·지역·키워드·핵심 작가·SAB DB 경매기록 기반 대표 작품 1~4점 정리.

운용 방식

S-1 화풍 모듈 가로 스와이프 카드 1장 = 본 데이터셋 화풍 1건. 답변에서 질의 화풍을 첫 카드(보더 강조 2px indigo)로 배치 + 관련 화풍을 뒤이어 노출. 카드 터치 시 "[화풍명]에 대해 알려줘" 자동 질의.

대표 작품 썸네일은 SAB DB의 실제 경매기록(낙찰 작품)에서 추출. 작품 1~4점 가로 표시 + 작가·작품명·연도 오버레이. 본 mockup은 그라데이션으로 색감만 표현.

큐레이터 검토 필요 항목

  • 대표 작품 1~4점 선정 기준: SAB DB 경매기록 중 (a) 낙찰가 상위 (b) 미술사 정착도 (c) 시각 임팩트 균형. 큐레이터 우선순위 룰 확정 필요.
  • 작품 이미지 권리: SAB가 경매·거래에서 사용한 작품 이미지 활용. 저작권 만료 작품 우선, 현존 작가 작품은 SAB 경매 카탈로그 게재본 활용.
  • 작가-화풍 매핑(N:M): 한 작가가 여러 화풍에 속할 수 있음(예: 박서보 = 한국 앵포르멜 + 단색화). 답변 시 우선 표시 화풍 룰 큐레이터 확정 필요.
  • 한국화 신경향·뉴미디어·동시대 추상: 흐름 명확성·작가 표본 보강 필요.
  • YBA·일본 네오 팝: 그룹/사조 경계 모호. 큐레이터 정의 필요.
